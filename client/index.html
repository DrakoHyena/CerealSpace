About

Cereal Space is a fast, pure js, entity management system that aims to "solve" as much of the common technical boilerplate for web games as you like. As such, you can expect most things from collisions and networking to already be done for you out of the box. The intended use is copy whatever parts of the engine you would like and to edit it inside of your own project. NOTE: More info in the performance section. Understanding a few key constraints is vital in order to utilize this engine fully.
