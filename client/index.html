About

Cereal Space is a fast, pure js, entity management system that aims to "solve" the common technical hurdles that can hinder indie webgame developement. As such, you can expect most things from collisions and networking to already be done for you and should be as fast as possible out of the box. However, you should still at least have an idea of how things work, because you are expected to use Cereal Space as a template to build off of, aiding in performance.

This engine works off of storing entities as data in an ArrayBuffer. This allows us to have granular control over the memory and the ability to quickly manipulate it, which is why so many entities can be handled at once. This has the upside of making network signifigantly cheaper than it would be otherwise since the data is already in a sorted ArrayBuffer.
