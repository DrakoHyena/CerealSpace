About

Cereal Space is a fast, (mostly) pure js, entity management system that aims to "solve" as much of the common technical boilerplate for web games as you like. As such, you can expect most things from collisions and networking to already be done for you out of the box. The intended use is copy whatever parts of the engine you would like and to edit it inside of your own project. NOTE: More info in the performance section. Understanding a few key constraints is vital in order to utilize this engine fully.

"Mostly pure js?" you ask? Initally the project was pure js because I wanted it to be as accesible as possible so that anyone can tear into it. However, despite my best efforts, javascript won... specifically in sorting. This portion of code was very frigaile and there was little reason to mess with it, so I considered the tradeoff worth it here. That said, you can always swap it out with your own solution.
